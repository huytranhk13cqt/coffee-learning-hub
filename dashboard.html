🔒 Security Measures

Server Security

  • ✓ CSP headers (default-src 'none', frame-ancestors 'none')
  • ✓ @fastify/helmet enabled
  • ✓ Rate limiting (20/min on exercise submit)
  • ✓ UUID validation on X-Session-Id header
  • ✓ postgres.js tagged templates (SQL injection proof)
  • ✓ correct_answer NEVER in GET responses
  • ✓ INNER JOIN on results (no answer leakage)
  • ✓ pg_advisory_xact_lock (race condition guard)
  • ✓ @fastify/compress (gzip > 1KB)
  • ✓ X-Request-Id propagation

Client Security

  • ✓ DOMPurify on Mermaid SVG output
  • ✓ No sensitive data in localStorage
  • ✓ AbortSignal.any() for timeout + navigation
  • ✓ No Content-Type on GET requests
  • ✓ WCAG 2.1 AA compliance (contrast, landmarks, aria)
  • ✓ React Compiler auto-memoization
